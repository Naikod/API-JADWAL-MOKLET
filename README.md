## HOW TO USE THE PROGRAM?

 You need these
 1. Create your file
 2. Open CMD/Bash and select the location your new file
 3. Write code below on your CMD
```git
npm init
npm i node-fetch@2.6.1
npm i readline-sync
git clone https://github.com/Naikod/API-JADWAL-MOKLET
cd API-JADWAL-MOKLET
node index
```
